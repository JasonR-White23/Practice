Introduction

This webpage is a short, 2 step instructional guide to making a HTML webpage. It shows the bare minimum needed to structure a HTML webpage much like this one itself.

Step 1

Creating the Template for your HTML Document

The best place to start when creating a HTML document is with a template with the "bare bones" of the HTML document itself.

In the following order add these elements to your page, remember that most elements have an opening (<element>) and closing (</element>) tag that surrounds the content within your elemenets:

  1. <!DOCTYPE html>

    This line of code tells the computer what type of file it is reading when the file is opened. It is also a self closing tag, so it does not require a closing tag

  2. <html lang="">

    </html>

    This the <html> element, this element contains all the HTML elements you would like to include on your HTML page. The lang="" attribute is used to determine the language of your HTML page. (english = en)

  3. <head>
    <title>
    (Page title here)
    </title>
    </head>

    The <head> contains the information about your HTML page, such as the title of your page, a description of your page, and so on. Information is determined within the <meta /> tag (a self closing element).

  4. <body>
    (Page content here...)
    </body>

    The <body> element contains the content you want to display to the users of your page. This tag should be within the <html> element but not within the <head> element.

Example

"Bare Bones" of an HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8" />
<meta name="viewport" content="width=device-width, initialscale=1.0" />
<meta name="description" content="A webpage instructing how to create a HTML webpage" />
<title>
Bare Bones Example
</title>
</head>
<body>
<h1>
Bare Bones Example
</h1>
<p>
This is where you add the conent you want displayed on your page.
</p>
</body>
</html>
Step 2

Add your Wepages Content

The only thing left to do now, is to fill your page with content! I included two elements within the example above. The first element is a header element, there are six heading elemets beginning at the largest font size (<h1>) through to the smallest text (<h6>). The second element withing in the example is the paragraph element (<p>), this is used for writing paragraphs which is displayed as a block of text.
